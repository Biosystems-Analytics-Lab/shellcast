# ShellCast

ShellCast is a Python Flask web application that helps shellfish growers make harvesting decisions based on predicted rainfall. See [docs/PROJECT_OVERVIEW.md](docs/PROJECT_OVERVIEW.md) for full project overview, architecture, and documentation index.

## Documentation (docs directory)

- [docs/ANALYSIS.md](docs/ANALYSIS.md) – analysis scripts and cron setup
- [docs/CITATION.md](docs/CITATION.md) – how to cite and give attribution
- [docs/DATABASE.md](docs/DATABASE.md) – database access and management
- [docs/DEVELOPER.md](docs/DEVELOPER.md) – repo structure and development setup
- [LICENSE.md](LICENSE.md) – ShellCast license (CC BY 4.0)
- [docs/MANUAL_UPDATES.md](docs/MANUAL_UPDATES.md) – simple updates to ShellCast
- [docs/WEB.md](docs/WEB.md) – web pages and Google App Engine setup
